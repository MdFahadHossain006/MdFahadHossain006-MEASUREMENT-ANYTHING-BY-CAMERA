## ✨ Features

### 🎯 Core Features
- **Real-time Camera Measurement** - Measure objects instantly using your device camera
- **Smart Calibration** - Calibrate using any object of known size
- **High Accuracy** - Pixel-perfect measurement with adjustable reference
- **Cross-Platform** - Works on mobile, tablet, and desktop browsers

### 📱 User Experience
- **Glassmorphism UI** - Modern, premium glass-morphism design
- **Step-by-Step Guide** - Easy to follow instructions for beginners
- **Visual Feedback** - Real-time points, lines, and measurements on camera feed
- **Mobile Responsive** - Fully responsive design for all devices

### 📄 PDF Reports
- **Export Measurements** - Download complete measurement reports as PDF
- **Image Capture** - Automatically captures camera view with measurements
- **Detailed Information** - Includes calibration data, notes, and timestamps
- **Professional Format** - Clean, professional report layout
